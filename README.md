# EnergyMeter-API

## Install
You need energymeter-utils git repository (https://github.com/Integrity-Ltd/energymeter-utils.git)
Install it to same root directory of this project.
Then install this package:
```
npm install
```
## Run dev
```
npm run dev
```
## Run with ts-node
```
npx ts-node src/app.ts
```
